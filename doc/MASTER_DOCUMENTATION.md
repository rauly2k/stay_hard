## Executive Summary

**StayHard** is a Flutter-based motivation and discipline-building application that helps users establish and maintain positive habits through:

- **Template-based habit system** with 20 pre-defined habits across 5 profiles that will be recommended for user ( + custom habits that can be added by user)
- **AI-powered program matching** using Google Gemini 2.5 Flash
- **Hard-to-dismiss alarms** with 4 challenge types (Math, Typing, Custom Text, Simple)
- **Native Android app blocking** for distraction-free focus sessions
- **AI Notifications system** with custom selectable (predefined) archetypes and adjustable intensity - for daily notifications for motivations (related to user profile, goals, habits etc)
- **Goal tracking** with detailed analytics and progress visualization - habits can be linked to goals and vice-versa
- **Real-time cloud sync** using Firebase Firestore with local Hive caching
